# Fatwa CMS Bot | نظام إدارة وأرشفة الفتاوى

بوت تليجرام متطور لإدارة وأرشفة ونشر الفتاوى الشرعية، مع واجهة برمجية (API) ونظام نشر تلقائي.

## 🌟 المميزات
- **إدارة كاملة**: أرشفة الفتاوى مع التصنيفات، العلماء، والمصادر.
- **بحث ذكي**: محرك بحث سريع يدعم البحث بالنص، الرقم، العالم، أو التصنيف.
- **دعم الذكاء الاصطناعي**: تكامل مع Groq (Llama) للبحث الذكي والاستعلامات المتقدمة.
- **نشر تلقائي**: جدولة نشر الفتاوى في القنوات والمجموعات بشكل يومي أو أسبوعي.
- **واجهة API**: خادم FastAPI مدمج للوصول إلى البيانات برمجياً.
- **إحصائيات ومراقبة**: نظام تتبع للأداء ونشاط المستخدمين.

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.9 أو أحدث.
- مفتاح بوت من @BotFather.
- (اختياري) مفتاح API من Groq للبحث الذكي.

### خطوات التثبيت
1. قم بتحميل المستودع:
   ```bash
   git clone https://github.com/your-username/fatwa-cms-bot.git
   cd fatwa-cms-bot
   ```

2. إنشاء بيئة افتراضية وتفعيلها:
   ```bash
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # Linux/macOS
   source .venv/bin/activate
   ```

3. تثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```

4. إعداد المتغيرات البيئية:
   قم بنسخ ملف `.env.example` إلى `.env` وقم بملء البيانات المطلوبة:
   ```bash
   cp .env.example .env
   ```

### التشغيل
- **تشغيل البوت:**
  ```bash
  python main.py
  ```
- **تشغيل الـ API (في نافذة منفصلة):**
  ```bash
  uvicorn api.server:app --host 0.0.0.0 --port 8000
  ```

## 📁 هيكل المشروع
- `main.py`: المدخل الرئيسي لتشغيل البوت.
- `core/`: الإعدادات، الاتصال بقواعد البيانات، والأدوات العامة.
- `handlers/`: معالجات أوامر البوت (البحث، الإدارة، القنوات).
- `api/`: خادم FastAPI والواجهة البرمجية.
- `data/`: تخزين قواعد البيانات والنسخ الاحتياطي (مستبعد من Git).
- `assets/`: الوسائط والملفات التوضيحية.

## 🔒 الأمان والخصوصية
- جميع المفاتيح السرية يتم إدارتها عبر ملف `.env`.
- ملفات البيانات (`.db`) والنسخ الاحتياطي مستبعدة من التتبع لضمان سرية البيانات.
- يتم استخدام نظام Singleton Lock لضمان تشغيل نسخة واحدة فقط من البوت.
