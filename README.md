## 🌟 المميزات
- **إدارة كاملة**: أرشفة الفتاوى مع التصنيفات، العلماء، والمصادر.
- **بحث ذكي**: محرك بحث سريع يدعم البحث بالنص، الرقم، العالم، أو التصنيف.
- **دعم الذكاء الاصطناعي**: تكامل مع Groq (Llama) للبحث الذكي والاستعلامات المتقدمة.
- **نشر تلقائي**: جدولة نشر الفتاوى في القنوات والمجموعات بشكل يومي أو أسبوعي.
- **نظام الحماية**: حماية ضد الضغط المتكرر (Flood) وإدارة صلاحيات المسؤولين.
- **إحصائيات ومراقبة**: نظام تتبع للأداء ونشاط المستخدمين.

## 📁 هيكل المشروع
- `main.py`: المدخل الرئيسي لتشغيل البوت.
- `core/`: الإعدادات، الاتصال بقواعد البيانات (aiosqlite)، والأدوات العامة.
- `handlers/`: معالجات أوامر البوت مقسمة إلى حزم (fatwa, search, admin, channels).
- `data/`: المجلد المخصص لقواعد البيانات النشطة والنسخ الاحتياطي (مستبعد جزئياً من Git).
    - `data/backups/`: النسخ الاحتياطية (مستبعدة).
    - `data/fatwa_bot.db`: قاعدة بيانات الفتاوى (مُتتبعة).
- `logs/`: سجلات العمليات مع خاصية التدوير التلقائي.
- `scripts/`: سكريبتات التشغيل والصيانة.
- `assets/`: الوسائط والملفات التعريفية.

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.10 أو أحدث.
- مفتاح بوت من @BotFather.
- (اختياري) مفتاح API من Groq للبحث الذكي.

### خطوات التثبيت
1. قم بتحميل المستودع:
   ```bash
   git clone https://github.com/your-username/fatwa-cms-bot.git
   cd fatwa-cms-bot
   ```

2. إنشاء بيئة افتراضية وتفعيلها:
   ```bash
   python -m venv venv
   # Windows
   .\venv\Scripts\activate
   # Linux/macOS
   source venv/bin/activate
   ```

3. تثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```

4. إعداد المتغيرات البيئية:
   قم بنسخ ملف `.env.example` إلى `.env` وقم بملء البيانات المطلوبة (التوكن، آيدي المالك، إلخ).

### التشغيل
- **تشغيل البوت:**
  ```bash
  python main.py
  ```

## 🔒 الأمان والخصوصية
- جميع المفاتيح السرية يتم إدارتها عبر ملف `.env`.
- ملفات البيانات (`.db`) والنسخ الاحتياطية مستبعدة من التتبع لضمان سرية البيانات.
- يتم استخدام نظام **Singleton Lock** لضمان تشغيل نسخة واحدة فقط من البوت ومنع تضارب البيانات.
